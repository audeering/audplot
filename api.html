cepstrum¶

audplot.cepstrum(cc_matrix, hop_duration, *, channel=0, ax=None, cmap='magma')[source]¶

Cepstrum.

Parameters

  • cc_matrix (ndarray) – cepstral coefficients matrix with magnitude values

  • hop_duration (float) – hop duration in seconds

  • channel (int) – channel index

  • ax (Optional[Axes]) – axes in which to draw the plot

  • cmap (str) – color map

Return type

AxesImage

Returns

Image object

Example

>>> import librosa
>>> import matplotlib.pyplot as plt
>>> x, sr = librosa.load(librosa.ex('trumpet'))
>>> y = librosa.feature.mfcc(x, sr)
>>> hop_dur = 512 / sr  # default hop length is 512
>>> image = cepstrum(y, hop_dur)
>>> _ = plt.colorbar(image)
>>> plt.tight_layout()

confusion_matrix¶

audplot.confusion_matrix(truth, prediction, *, labels=None, percentage=True, ax=None)[source]¶

Confusion matrix between ground truth vs. predicted labels.

The confusion matrix is calculated by audmetric.confusion_matrix.

Parameters

  • truth (Union[Sequence, Series]) – truth values

  • prediction (Union[Sequence, Series]) – predicted values

  • labels (Optional[Sequence]) – labels to be included in confusion matrix

  • percentage (bool) – if True present the confusion matrix with percentage values instead of absolute numbers

  • ax (Optional[Axes]) – axes in which to draw the plot

Example

>>> truth = ['A', 'A', 'A', 'B', 'B', 'B', 'C', 'C', 'C']
>>> prediction = ['A', 'A', 'B', 'B', 'C', 'C', 'A', 'A', 'C']
>>> confusion_matrix(truth, prediction, percentage=False)

>>> confusion_matrix(truth, prediction, labels=['A', 'B', 'C', 'D'])

spectrum¶

audplot.spectrum(magnitude, hop_duration, centers, *, channel=0, ax=None, cmap='magma')[source]¶

Plot spectrum.

Parameters

  • magnitude (ndarray) – matrix with magnitude values

  • hop_duration (float) – hop duration in seconds

  • centers (ndarray) – array with center frequencies

  • channel (int) – channel index

  • ax (Optional[Axes]) – axes to plot on

  • cmap (str) – color map

Return type

AxesImage

Returns

Image object

Example

>>> import librosa
>>> import matplotlib.pyplot as plt
>>> x, sr = librosa.load(librosa.ex('trumpet'))
>>> y = librosa.feature.melspectrogram(x, sr, n_mels=40, fmax=4000)
>>> y_db = librosa.power_to_db(y, ref=np.max)
>>> hop_dur = 512 / sr  # default hop length is 512
>>> centers = librosa.mel_frequencies(n_mels=40, fmax=4000)
>>> image = spectrum(y_db, hop_dur, centers)
>>> _ = plt.colorbar(image, format='%+2.0f dB')
>>> plt.tight_layout()
